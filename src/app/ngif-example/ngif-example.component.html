In conclusion we found out that there are many ways of conditionaly rendering data. If we have a large component tree we would rather use *ngIf instead of hidden because that is the better approach of doing it. While hidden the component is still being created and it is just now being displayed to the user, eveything in the back runs and there is no need for that. That is why it is better to use *ngIf.
